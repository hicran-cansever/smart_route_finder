# Smart Route Navigator

This project finds the shortest path between two points on a university campus map using **Dijkstra's Algorithm**.

## How to Use
1. Open `index.html` in your browser (use Live Server for best results).
2. Click on any blue node to set the **Start Point**.
3. Click on another blue node to set the **End Point**.
4. The application will draw a **red line** showing the shortest path.
5. Click "Clear Selection" to start over.

## Technologies
- JavaScript
- Leaflet.js (Map Library)
- HTML & CSS
